Placas Arduino

Las placas Arduino son el corazón del sistema. Las más comunes son: Arduino UNO, Mega, Nano y Leonardo.

Sensores

Existen múltiples sensores compatibles con Arduino: temperatura, humedad, ultrasonido, movimiento, entre otros.

Alimentación

Arduino puede alimentarse por USB (5V), pilas, baterías de litio o fuentes externas de hasta 12V.

Entorno de Programación (IDE)

El Arduino IDE permite escribir, compilar y cargar programas en la placa. El lenguaje usado es C/C++ con librerías propias.


// Ejemplo: Parpadeo LED
int led = 13;
void setup() {
  pinMode(led, OUTPUT);
}
void loop() {
  digitalWrite(led, HIGH);
  delay(1000);
  digitalWrite(led, LOW);
  delay(1000);
}

Ejemplo Interactivo: LED Virtual

Haz clic en el botón para encender y apagar el LED, igual que en Arduino.
